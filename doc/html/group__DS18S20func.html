int8_t DS18S20_SendFunctionCmd ( TSDS18S20 *  pDS18S20,
uint8_t  funCommand 
)

sends function commands to DS18S20.

Parameters
[in,out]pDS18S20pointer to the structure that represent DS18S20
[in]funCommandfunction command to be send.
Returns
int8_t depending of the submited command, the function returns different values.
Return codes:
- In case of READ_SCRATCHPAD command, it returns: 0 if CRC error is detected, 1 for successful scratchpad read;
- For READ_POWER_SUPPLAY command it returns: 1 for external power supply, 0 for parasite powered sensor;
- For all other commnads the default return value is 1;
- returns -2 in case of unkown function command;
- returns -1 in case of failed OWReset().
